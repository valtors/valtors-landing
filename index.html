                        Why Relay
                        
One install surface. Full agent capability.

                        
Relay is built for teams that want local-first power without plugin sprawl. It feels like a terminal tool, not a toy integration.

                    

                    

                        

                            
Single Binary

                            
One go install or npx userelay. No Docker, no dependencies, no config files.

                        

                        

                            
40 Built-in Tools

                            
File, image, PDF, web, data, text, workflow. Everything an agent needs, pre-wired.

                        

                        

                            
Stdio + HTTP

                            
Works with Claude Desktop, Cursor, VS Code, any MCP client. Both transports, one binary.

                        Setup
                        
Running in 10 seconds

                        
Drop Relay into the MCP client you already use. No service mesh, no sidecar process, no local scripting ceremony.

                    

                    

                        

                            Claude Desktop · claude_desktop_config.json
                            
Local stdio transport

                            
Point Claude Desktop at Relay and it boots on demand.

                            
{
  "mcpServers": {
    "relay": {
      "command": "npx",
      "args": ["-y", "userelay"]
    }
  }
}

                        

                        

                            Cursor · .cursor/mcp.json
                            
Drop into Cursor

                            
Same contract, same binary, zero extra packaging.

                            
{
  "mcpServers": {
    "relay": {
      "command": "npx",
      "args": ["-y", "userelay"]
    }
  }
}

                        

                        

                            HTTP mode · any client
                            
Expose over HTTP

                            
Run Relay as a networked MCP endpoint when you need it.

                            
npx userelay --http :8080
