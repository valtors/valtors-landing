File ops, image processing, PDF, web, data transforms, workflow orchestration. Everything your agent needs ships in a single zero-config binary.

$ npx userelay

    Setup
    

        
Running in
ten seconds.

    

    

        

            
Claude Desktop

            
claude_desktop_config.json

            
{
  "mcpServers": {
    "relay": {
      "command": "npx",
      "args": ["-y", "userelay"]
    }
  }
}

        

        

            
Cursor

            
.cursor/mcp.json

            
{
  "mcpServers": {
    "relay": {
      "command": "npx",
      "args": ["-y", "userelay"]
    }
  }
}
