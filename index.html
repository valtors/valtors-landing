Built on solid principles
What we believe.

Open Source

MIT licensed. Free to use, modify, and build on. No paid tiers, no open-core, no tricks.

Research First

We analyzed 1665 developer discussions before building Observer. We listen before we build.

Community Driven

Built with feedback from developers like you. External contributors, external testers, real engagement.

Fast and Reliable

Go stdlib and SQLite. Boring tech on purpose. No dependencies you have to debug at 2am.

Security Hardened

Metadata-only by default. 0600 permissions. Session isolation. Redaction before persistence.

CLI Friendly

Works where you work. Terminal native. No dashboard to babysit. One binary, one env var.

Community First

Built in public. For everyone.
