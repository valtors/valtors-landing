what we built

six tools. one philosophy.

boring tech, zero dependencies where possible, built to outlast the hype cycle.

Relay
one MCP server, 40 tools, one binary. files, images, PDFs, web fetch, workflows. stdio and HTTP transport. zero runtime dependencies. other people install 5 servers to do what this does alone. npm: userelay.
GitHub
Reflow
responsive toolkit for TypeScript that doesn't break SSR. same API in React, Vue, Svelte, Solid, Qwik, Preact, Angular. no hydration mismatch warnings. no typeof window checks. no framework lock-in. npm: usereflow.
GitHub
Observer
your agent makes 30 tool calls. you see the final text. observer logs every call to SQLite so you can see what actually happened. SSE transport, tool filtering, trace replay. it sits between your client and the server and watches. silently.
GitHub
mcprobe
security scanner for MCP servers. finds prompt injection in tool descriptions, tool shadowing, resource exposure to sensitive paths, baseline drift. your MCP server is probably vulnerable. you connected it to your agent without checking. this tells you what you missed. single Go binary.
GitHub
Vault
run untrusted MCP tools without burning down your machine. strips secrets from environment, blocks sensitive file paths, scans for prompt injection in tool descriptions, logs every action to SQLite. not a real sandbox. an audit layer. honest about what it is.
GitHub
Pulse
connect everything. your ai does the rest. github, gmail, calendar in one place. asks "what did i miss" and gets a real answer. 38 notifications become 3 urgent items. remembers what matters, forgets what doesn't. local-first, single binary, your data on your machine. not a framework. a product.

why

everything you need. nothing you don't.

Open source

MIT licensed. free to use, modify, and build on. no proprietary hooks, no vendor lock-in, no telemetry. if you find telemetry in here, it's a bug.

Local first

single binary, SQLite storage, Go stdlib. your data doesn't leave your machine. boring tech on purpose because boring tech doesn't break at 3am.

MCP native

built for the Model Context Protocol from day one. stdio transport, stdlib JSON-RPC. no magic, no abstraction layers, no frameworks on top of frameworks.

Well documented

clear docs, working examples, guides for every step. you shouldn't have to read source code to understand the API. if you do, that's a bug too.

Security first

mcprobe finds prompt injection, tool shadowing, resource exposure. observer redacts sensitive data by default. security is built in. not bolted on after someone files an issue.

Community driven

built with feedback from developers. contributions welcome. external contributors shipping. no corporate backing, no VC money, no roadmap dictated by a sales team.
