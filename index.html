Products

Modular. Open. Designed for real-world workflows.

Three tools, one philosophy. Boring tech, zero dependencies where possible, built to last.

Relay
One MCP server for files, images, PDFs, and more. 40+ tools exposed through a single Go binary. Stdio transport, zero config.
GitHub
Reflow
SSR-safe, responsive toolkit for TypeScript. Responsive primitives that work on the server, in the browser, and everywhere in between. htm, no build step.
GitHub
Observer
Transparent MCP proxy for agent observability. SQLite traces, metadata-only by default, redaction patterns, session isolation. See what your agents actually do.

Built for developers

Everything you need. Nothing you don't.

Open source

MIT licensed. Free to use, modify, and build on. No proprietary hooks, no vendor lock-in, no telemetry.

Local first

Human-in-the-loop always. Single binary, SQLite storage, Go stdlib. Boring tech on purpose.

MCP native

Built for the Model Context Protocol from day one. Stdio transport, stdlib JSON-RPC, no magic.

Well documented

Clear docs, working examples, and guides for every step. No reading source code to understand the API.

Security hardened

DB permissions locked down, metadata-only traces by default, redaction patterns, session isolation built in.

Community driven

Built with feedback from developers like you. Contributions welcome, external contributors shipping.
